Improves the [mcpmarket.com](https://mcpmarket.com/search) search experience.

### Features

- Compact sticky search bar (search, type tabs, favorites, filter)
- Soft search while typing (no full-page reload / skeleton flash)
- Category filter dropdown (keeps `category_slug` + `type` in the URL)
- Heart favorites on result cards, persisted with `GM.setValue`
- Favorites mode: browse saved items; scroll position preserved both ways
- Infinite scroll (replaces "Load More")
- Left-click opens a result in a **background** tab (`GM.openInTab`)

### Install

Click **Install this script**, then open [mcpmarket.com/search](https://mcpmarket.com/search).
